## Food Delivery App built on Next.js with React + Redux, Node.js, and Paypal for payments
- Frontend built with Next.js + React and Redux
- Backend built with Node.js and MongoDB

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
